# ibot2 for TilesSurvive

### Ускорь рутину в игре и сосредоточься на результате

`ibot2` - Windows-бот для **TilesSurvive**, который берет на себя повторяющиеся действия и помогает играть стабильнее и эффективнее.

## Почему ibot2

- Экономит время на однотипных игровых действиях.
- При активных рейдах в союзе фармит незаконно много ящиков ускорений. 
- Запускается одним файлом `ibot2.exe` без установки.
- **Не требует** никаких данных от учетных записей.
- Показывает предупреждение о обновлении и предлагает скачать свежую версию.

## Что умеет бот

- **Навигация**: автоматизация маршрутов и перемещений.
- **Рейды**: автоматическое вступление в рейды.
- **Руководства по навыкам**: автоматический сбор "книг".
- **Конверты**: автоматический сбор конвертов при появлении.
- **Помощь**: автоматическое прожатие помощи (ручек).
- **Сбор угля**: автоматический сбор угля в шахте раз в час.
- **Лечение войск**: автоматическое лечение войск.
- **Автоатака зараженных**: автоматически бьет зомби, пока не закончится энергия.
- **Автосоздание рейдов на Альф**: автоматически создает рейды, пока не закончится энергия.
- **Автопереподключение**: при вылете из аккаунта (если запустился на другом устройстве).

## Скачать и начать за 5 минут

1. Скачай официальный клиент Tiles Survive 
2. Зайди в аккаунт через официальный клиент игры. 
3. Скачай `ibot2.exe` из последнего релиза.
4. Запусти файл.
5. Выбери нужные функции. 
6. Нажми "Старт".

## Системные требования

- **Windows 10/11**
- Права администратора (для корректной работы управления курсором)

## Частые вопросы

### Появилось сообщение про неподходящую версию лицензии

Лицензия не блокируется, но рекомендуется обновить клиент.  
Используй кнопку **"Скачать новую версию"** в окне активации.

### Бот не запускается

- Запусти `ibot2.exe` от имени администратора.
- Убедись, что антивирус не заблокировал файл.
- Скачай свежую версию из последнего релиза.

## Обновления

Актуальные версии и обновления публикуются в разделе **Releases**.  

Для загрузки последней версии откройте страницу релиза и скачайте нужный файл оттуда.
